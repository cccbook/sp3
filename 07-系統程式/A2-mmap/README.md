CONFORMING TO
       POSIX.1-2001, POSIX.1-2008, SVr4, 4.4BSD.

       On POSIX systems on which mmap(), msync(2), and munmap() are
       available, _POSIX_MAPPED_FILES is defined in <unistd.h> to a
       value greater than 0.  (See also sysconf(3).)
