# IR 中間碼

```
00: t1 = 0
01: s = t1
02: t2 = 1
03: i = t2
04: (L1)
05: t3 = i
06: t4 = 10
07: t5 = t3 <= t4
08: ifnot t5 goto L2
09: t6 = s
10: t7 = i
11: t8 = t6 + t7
12: s = t8
13: t9 = i
14: t10 = 1
15: t11 = t9 + t10
16: i = t11
17: goto L1
18: (L2)
```

```
05: ld8 r4 +10
06: r5 = r3<=r4
07: r5 = r245r0

09: ld8 r7 [ds+4]
10: r8 = r6+r7
11: st8 r8 [ds+5]

```
