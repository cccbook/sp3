## 使用

http 下載網頁: 執行後會多出一個 index.html 檔案

```
$ wget http://www.example.com/
```

ftp 下載檔案: 執行後會多出一個 wget-latest.tar.gz 檔案

```
$ wget ftp://ftp.gnu.org/pub/gnu/wget/wget-latest.tar.gz
```

http 砍站: 執行後會多出一個 www.example.com 資料夾，裡面有該站的網頁。

```
wget -r -l 0 http://www.example.com/
```
