# conan

* https://docs.conan.io/en/latest/getting_started.html

* https://github.com/ttroy50/cmake-examples/tree/master/07-package-management/D-conan

## install

```
$ pip install conan
```

## make

```
   18  mkdir build && cd build
   19  conan install ..
   20  cmake .. -G "Visual Studio 16"
   21  cmake --build . --config Release
   22  ./bin/md5
```
