# LLVM

## Ubuntu 安裝

* https://apt.llvm.org/

For convenience there is an automatic installation script available that installs LLVM for you.
To install the latest stable version:

```
bash -c "$(wget -O - https://apt.llvm.org/llvm.sh)"
```
