## qemu 載入系統 -- 以 xv6 為例

```
$ cd img

$ qemu-system-i386 -nographic -drive file=fs.img,index=1,media=disk,format=raw -drive file=xv6.img,index=0,media=disk,format=raw -smp 2 -m 512
```
